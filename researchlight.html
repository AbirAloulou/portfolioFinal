Abstract:

This memory was realized at Re-searchlight company as part of my final year internship. The goal is to design a data warehouse to analyze user profiles and behaviors in the medical field, using the "Open Source Intelligence" (OSINT) technique. More precisely, our approach consists of four steps. First, we collected data from Facebook group members using Phantom Buster. Then, we performed a cleaning step to ensure data consistency and reliability in order to generate the multidimensional schema of the data warehouse. Then, we defined the loading procedures of this processed data, in order to be exploited through a set of analytical tools.
